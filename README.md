# Customer Churn Classification

This repository contains a machine learning artifact created through the
Break Through Tech AI Program.

## Project Overview

In this project, I worked with telecommunications customer data to explore
customer churn prediction using machine learning.

The project focused on understanding how classification models learn from
customer features and predict whether a customer is likely to discontinue
a service.

## Skills Practiced

- Python
- Pandas
- NumPy
- Scikit-learn
- Data preparation
- Feature and label selection
- Train-test splitting
- K-nearest neighbors classification
- Cross-validation
- Model evaluation

## Machine Learning Process

During the project, I practiced:

1. Examining and preparing customer data
2. Separating predictive features from the churn label
3. Creating training and testing datasets
4. Training a classification model
5. Generating predictions
6. Evaluating model performance
7. Using cross-validation to measure model consistency
8. Comparing different model parameters
